# Reability Landing Page Tecnológica

Site estático em HTML/CSS/JS, com imagens incorporadas no próprio HTML.
Abra `index.html` no navegador.

Inclui:
- Splash narrativo
- Fundo canvas com partículas neurais
- Scroll reveal
- Cards 3D
- Accordion de especialidades
- Métricas animadas
- Seção da Denise Neves
- CTA e footer institucional
